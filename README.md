## 安裝和運行

1.克隆倉庫：
   ```bash
   git clone https://github.com/xuan080731/translation-chatbot.git

2.進入專案目錄：
bash

cd translation-chatbot

3.安裝依賴：
bash

pip install -r requirements.txt

4.設置 .env 文件：
在專案根目錄下創建一個 .env 文件，並添加你的 Google API 密鑰：

GOOGLE_API_KEY=你的API密鑰

5.運行程式：
bash

python translation_chatbot.py

6.使用聊天機器人：
程式運行後，會提示你輸入目標語言（例如 "Chinese"）。
然後，你可以輸入要翻譯的文本，程式會顯示翻譯結果。
輸入 "exit" 退出程式。
